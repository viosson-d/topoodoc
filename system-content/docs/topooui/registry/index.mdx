---
title: "Registry"
description: "Run your own code registry."
---
You can use the `shadcn` CLI to run your own code registry and distribute custom components, hooks, pages, config, rules, and other files to any project.

<Callout>
**Note:** The registry works with any project type and any framework, and is not limited to React.
</Callout>
